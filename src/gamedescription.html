But du jeu :

Le but du jeu est de trouver toutes les paires de cartes identiques. À chaque tour, vous devez retourner deux cartes et essayer de les faire correspondre.

Règles du jeu :

  • Retourner deux cartes à la fois : Lors de chaque tour, le joueur retourne deux cartes. Les cartes sont comparées pour vérifier si elles correspondent.
  • Paire ou pas de paire : Si les deux cartes retournées sont identiques (une paire), elles restent face visible. Si elles ne correspondent pas, elles sont retournées face cachée après un court délai.
  • Mémoire : Le jeu repose sur la mémoire. Il faut se souvenir de l'emplacement des cartes déjà retournées pour les associer plus facilement.
  • Si le timer atteint 20 secondes, le jeu se termine automatiquement. Vous devrez avoir terminé avant que le temps ne soit écoulé !
